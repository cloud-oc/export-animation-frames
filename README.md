# 导出动画图层帧

Aseprite 插件 —— 选中图层，一键导出时间轴上所有帧为独立图片文件。

安装后在 **文件 → 导出 → 导出图层帧...** 中使用。

## 安装

1. 将本仓库打包为 `.zip`（需包含 `package.json` 和 `export-animation-frames.lua`）
2. 将扩展名改为 `.aseprite-extension`
3. 双击安装，或在 **编辑 → 首选项 → 扩展 → 添加扩展**

## 使用

1. 在 Aseprite 中选中要导出的图层（支持多选）
2. **文件 → 导出 → 导出图层帧...**
3. 选择导出范围、配置参数、点击"导出"

## 图层选择

| 选项         | 说明                       |
| ------------ | -------------------------- |
| 已选图层     | 导出时间轴中当前选中的图层 |
| 所有可见图层 | 导出所有可见图层           |
| 所有图层     | 导出全部图层               |
| 具体图层名   | 导出指定的单个图层         |

## 导出选项

| 选项       | 说明                                                                        |
| ---------- | --------------------------------------------------------------------------- |
| 输出位置   | 导出目录和格式（在文件管理器对话框中，通过“保存类型”下拉框选择导出格式）       |
| 文件名格式 | `{mapname}` 文件名 · `{layername}` 图层名 · `{fps}` 帧率 · `{frame}` 帧序号 |
| 导出缩放   | 1x ~ 10x                                                                    |
| 跳过空帧   | 跳过该图层上没有内容的帧                                                    |
| 去除重复帧 | 跳过与前一帧完全相同的帧，帧率自动重算                                      |
| 裁切到内容 | 裁切掉透明区域                                                              |
